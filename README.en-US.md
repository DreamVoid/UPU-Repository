# UPU-Repository

[简体中文](README.md) | [English](README.en-US.md)

This is the configuration repository for [UniversalPluginUpdater](https://github.com/DreamVoid/UniversalPluginUpdater), maintained by the UPU developer (that's me).

You are free to clone this repository and maintain your own repository according to the configuration template. There is only one restriction: you cannot claim that your self-maintained repository is part of UPU.

## 贡献配置

To contribute new plugin configurations, please see the [Contribution Guidelines](.github/CONTRIBUTION.en-US.md).
